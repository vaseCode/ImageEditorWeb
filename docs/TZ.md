# Мини-ТЗ: IEW studio

## 1. Название
ImageEditorWeb studio (Твеб-приложение для редактирования изображений)

## 2. Цель
Программа для обработки и редактирования изображений, с поддержкой слоёв, фильтров и инструментов

## 3. Пользователь
Обычный пользователь ПК (студент).

## 4. Функциональные требования (FR)
- FR1: Загрузка и редактирование изображений
- FR2: Многослойная работа с режимами наложения
- FR3: Инструменты: кисти, ластик, выделение, заливка и прочие
- FR4: Фильтры и коррекции: яркость, контраст, размытие
- FR5: Сохранение в PNG, JPG, WEBP

## 5. Нефункциональные требования (NFR)
- Приложение запускается без ошибок
- Понятные сообщения об ошибках (например: “Выделение не возможно”)

## 6. Модель данных (черновик)
**Изображение**
- Id (int)
- Title (string, not null)
- ImageData (image/BLOB)

## 7. Критерии приемки (как проверить)
- Можно импортировать/экспортировать изображения
- Можно редактировать изображения
- Можно отменить/повторить действие
