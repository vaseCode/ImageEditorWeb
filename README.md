# IEW studio

Автор: Дорофеев Василий, Илья Великанов, 25ИС11д

## Описание
*ImageEditorWeb studio* - это веб-приложение для редактирования изображений прямо в браузере. Аналог Photoshop с поддержкой слоёв, фильтров и основных инструментов

## Возможности приложения
- Система слоёв
- Инструменты (кисти, ластик, выделение, заливка и прочие)
- Работу с историей (undo/redo)

## Стек
C# / ASP.NET Core Web API / Blazor WebAssembly / Visual Studio

## Как запустить
1) Открыть solution в Visual Studio
2) Нажать F5 (Run)
